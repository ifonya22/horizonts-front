# horizonts-front
### Запуск сервисов:
```bash
# Первый раз для сборки
docker compose up --build 
# Последующие разы
docker compose up
```
### Остановка сервисов:
```
docker compose down
```

### Ссылки:
```

бэкенд http://localhost:8000/
база данных http://localhost:8080/
```
